# Design Sticks on 66

> **⚠️ Beta Notice**: This website is currently in active development. Many features are not yet functional, and placeholder content is used throughout.

## About

**Sticks on 66** is a celebration of 66 public golf courses located along the historic Route 66. This website showcases a unique golf trail spanning 8 states from Chicago to Los Angeles, blending luxury golf resort aesthetics with vintage Americana nostalgia.

**Target Launch Date**: July 4, 2026 — coinciding with America's 250th birthday and Route 66's 100th anniversary.

## Features

The website includes the following pages:

- **Home** - Landing page with animated statistics, state cards, featured courses carousel, and anniversary celebration
- **The Trail** - Detailed Route 66 trail information and FAQ
- **Tee Times** - Golf booking interface with search and filtering
- **Trail Membership** - Membership tiers and benefits program
- **Tournaments** - Tournament listings and information
- **Food & Beverage** - Dining venues along the trail
- **Shop** - E-commerce for golf merchandise and trail exclusives
- **Contact** - Contact form and communication hub
- **Oklahoma Courses** - Directory of 30+ Oklahoma golf courses

## Tech Stack

- **React 18** with TypeScript
- **Vite** - Build tool
- **Tailwind CSS v4** - Custom Route 66 color palette
- **Framer Motion** - Advanced animations (parallax, counters, stagger effects)
- **Radix UI** - Component library
- **React Router** - Client-side routing
- **Vercel** - Deployment platform

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The development server runs on `http://localhost:3000`.

## Beta Status

This project is in early development (v0.1.0). Current limitations include:

- All images are placeholder stock photos from Unsplash
- Form submissions are not connected to a backend
- Booking and payment processing is not functional
- Some buttons and links are placeholder (`href="#"`)
- Mock data is used for tournaments, products, and leaderboards
